# 🏨 Hotel Review Sentiment Analysis

## 📌 Project Overview
Customer reviews play a crucial role in understanding hotel service quality and guest satisfaction.  
This project analyzes large-scale hotel review data to uncover customer experience insights using data analysis and Natural Language Processing (NLP).

The analysis focuses on identifying factors influencing hotel ratings and extracting sentiment from textual reviews.

---

## 📊 Dataset
Hotel Review Dataset sourced from Kaggle containing:
- Customer review text
- Overall ratings
- Service, cleanliness, location, and value scores

Dataset Link: https://www.kaggle.com/

---

## 🛠️ Tools & Technologies
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- WordCloud
- TextBlob (Sentiment Analysis)
- Jupyter Notebook

---

## 🔎 Project Workflow
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Rating Distribution Analysis
- Feature Correlation Analysis
- Text Mining & Word Cloud Visualization
- Sentiment Analysis of Customer Reviews

---

## 📈 Key Insights
- Majority of hotel reviews show positive sentiment.
- Cleanliness and location strongly impact overall ratings.
- Customers providing longer reviews tend to give detailed service feedback.
- Sentiment analysis validates rating-based satisfaction trends.
